# Wheat Yield Predictor (Прогнозатор урожайності пшениці)

![C++](https://img.shields.io/badge/C++-solutions-blue?logo=c%2B%2B&logoColor=white)
![Makefile](https://img.shields.io/badge/Makefile-automation-brightgreen?logo=gnu&logoColor=white)

Цей проект являє собою консольну програму на C++ для прогнозування врожайності пшениці на основі різних агрономічних та кліматичних факторів.

## Основні можливості
Програма дозволяє користувачу обрати ключові параметри, які впливають на ріст та розвиток пшениці, і на основі цього розраховує орієнтовну врожайність. 

Враховуються такі фактори:
- **Сорт пшениці:** (наприклад, Подолянка, Краєвид, Скаген), кожен з яких має свої базові показники врожайності та стійкості.
- **Тип ґрунту:** Чорнозем, Суглинок, Піщаний, Лучний (з відповідними коефіцієнтами родючості).
- **Добрива:** Вплив різних типів добрив (NPK, Амофос, Карбамід або без добрив).
- **Агрокліматичний регіон:** Полісся, Лісостеп, Степ, Південний степ, Карпати.
- **Погодні умови по сезонах:** Можливість задати погоду для весни, літа, осені та зими (дощова, спекотна, помірна, суха, м'яка, вітряна).

## Структура проекту
Проект має модульну структуру: заголовні файли розміщені в директорії `cls/`, а файли реалізації — у `src/`.

```text
.
├── cls/                     # Заголовні файли (класи)
│   ├── FertilizerEffect.h   # Вплив добрив
│   ├── Region.h             # Агрокліматичний регіони
│   ├── SoilType.h           # Типи ґрунту
│   ├── VarietyBonus.h       # Додаткові характеристики (бонуси) сортів
│   ├── WeatherCondition.h   # Погодні умови
│   ├── WheatVariety.h       # Сорти пшениці
│   └── WheatYieldPredictor.h# Головний клас для розрахунку прогнозу
├── src/                     # Файли реалізації
│   ├── FertilizerEffect.cpp
│   ├── Region.cpp
│   ├── SoilType.cpp
│   ├── WeatherCondition.cpp
│   ├── WheatVariety.cpp
│   └── WheatYieldPredictor.cpp
├── main.cpp                 # Головний файл програми з інтерактивним меню
└── Makefile                 # Скрипт збирання
```

## Вимоги до системи
- Компілятор C++ з підтримкою стандарту **C++17** (наприклад, `g++` або `clang++`).
- Утиліта `make` для автоматизації збирання.

## Збирання та запуск

1. Відкрийте термінал у кореневій папці проекту.
2. Виконайте команду для компіляції:
   ```bash
   make
   ```
3. Запустіть скомпільовану програму:
   ```bash
   ./wheat_predictor
   ```

Якщо ви бажаєте підключити інший файл `main.cpp` з власною логікою, зберіть проект так:
```bash
make MAIN=/шлях/до/вашого/main.cpp
```

## Очищення проекту
Щоб видалити тимчасові об'єктні файли (`.o`) та згенерований виконуваний файл, виконайте:
```bash
make clean
```

## Використання
Після запуску програми використовуйте інтерактивне меню в консолі. Вам буде запропоновано покроково обрати:
1. Сорт пшениці.
2. Тип ґрунту.
3. Тип добрива.
4. Агрокліматичний регіон України.
5. Погодні умови для кожного з чотирьох сезонів (можна обрати кілька умов для одного сезону, вводячи їх номери через пробіл, а потім `0` для переходу далі).

В кінці програма проаналізує введені дані та виведе детальний розрахунок і прогноз урожайності.
